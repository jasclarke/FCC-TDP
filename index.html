Introduction

The Footy University webpage is a responsive product landing page template. It contains the following five sections:

  1. Home - Attention Grabber Section.
  2. Features - Highlights main features of the product.
  3. How It Works - Section describing and depicting how the product functions.
  4. Pricing - Section displaying the different pricing options for the product.
  5. Get It - Section with a form to allow a visitor to make contact.
It was built using vanilla HTML and CSS by Jason Clarke and is free to use for both commercial and personal use.

Prerequisites

Modification of the Footy University template requires knowledge of the following technologies:

  • HTML5
  • CSS3

Menu

The menu section contains links to each section of the landing page. Additionally, its position property is sticky making it easily accessible to the user regardless of which section of the page they are viewing. Subsequently, it has a designated area for a logo where you can replace the current one with yours by locating the piece of code below in the page and modifying the source attribute of the image tag. 

    
        <div><a href="#home"><img src="images/logo.png" alt="Footy University Logo" id="header-img"/></a></div>

If you have more sections then you can simply add links to them by locating the code below in the page and adding the anchor tags as reqired. Additionally, you can modify the other anchor tags as desired just remember to modifying the section(s) id attribute 

    
        <nav id="nav-bar">
        <a href="#features" class="nav-link">Features</a>
        <a href="#how-it-works" class="nav-link">How It Works</a>
        <a href="#pricing" class="nav-link">Pricing</a>
        <a href="#notify" class="nav-link">Get It</a>
        </nav>
